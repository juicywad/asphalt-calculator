# asphalt-calculator
An easy to use adjustable tonnage calculator for the asphalt industy and the like.
You enter your lenght and width and thickness and it does the rest for you. There is an adjustable denstiy slider that goes from 110 to 160 for more acurate calculations. There is also and option for calculating the tonnage of a circle. 
The last five calculations are saved in a history tab and you can add calcutlations to a job name and keep track of what your doing effortlessly.
It does all this offline for those jobs where service is out of the question.
